# swift-demo-habit-tracker

Swift / SwiftUI 학습을 위한 **습관 트래커(Habit Tracker)** 데모 앱입니다.
프론트엔드 개발자가 iOS 개발을 처음 익히는 것을 목표로, 작게 시작해 단계적으로 기능을 확장합니다.

> 서버 없이 **기기 한 대 안에서만** 동작하는 앱입니다.

## 무엇을 만드나요?

매일 반복하는 습관(예: 물 마시기, 운동, 독서)을 등록하고 관리하는 iOS 앱입니다.

## 기술 스택

| 항목 | 선택 |
|------|------|
| 언어 | Swift |
| UI | SwiftUI |
| 상태 관리 | `@State` / `@Observable` |
| 개발 도구 | Xcode |
| 최소 버전 | iOS 17+ |
| 외부 의존성 | 없음 |

## 학습 로드맵

| 단계 | 내용 | 새로 배우는 것 |
|------|------|---------------|
| **1단계** | 습관 목록 + 추가/수정/삭제 (메모리 상태만) | SwiftUI 기초, 상태 관리, 리스트, 화면 이동, 입력 폼 |
| 2단계 | 기기에 저장 (껐다 켜도 유지) | 로컬 영속 저장 |
| 3단계 | 오늘 체크 + 연속 달성일(streak) | 날짜 다루기, 파생 상태 |
| 4단계 | 상세 화면 + 달력 + 통계 | 차트, 복합 화면 |
| 5단계 | 매일 알림(리마인더) | 로컬 푸시 알림 |
